Interactive search rediscovered

Here’s a helpful tip I discovered while searching interactively (using C-s).

If you want to see all occurrences of your current search term, press M-s o. This will open a new buffer listing all the relevant instances. You can also conveniently extend your search by selecting additional words with C-w.

By using the sequence C-s C-w C-w M-s o, you can select two words starting from the current cursor position and view all their occurrences in the current buffer.

A fantastic way to streamline this process is by utilizing the ‘anything’ functionality. Try using anything-occur, and you might be amazed by the results!
